I specialize in evaluating AI-generated code and technical responses — analyzing them across multiple dimensions to determine which solutions are genuinely stronger and why. I help AI models understand what real-world software engineering quality actually looks like.

### 🎯 What I Evaluate

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║  ✅ Correctness     — Does it solve the actual problem?     ║
║                       Are edge cases handled properly?      ║
║                                                              ║
║  ✅ Clarity         — Is the code readable and intuitive?   ║
║                       Would a teammate understand it fast?  ║
║                                                              ║
║  ✅ Efficiency      — Time & space complexity considered?   ║
║                       Is the optimal approach being used?   ║
║                                                              ║
║  ✅ Reasoning       — Does the explanation hold up?         ║
║                       Is the logic sound and justified?     ║
║                                                              ║
║  ✅ Engineering     — Best practices followed?              ║
║     Quality          Appropriate design patterns applied?   ║
║                       Would this survive a real code review?║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```
